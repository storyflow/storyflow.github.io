三、数据类型

string：二进制安全的字符串
Lists：按插入顺序排序的字符串元素的集合。
Sets: 不重复且无序的字符串元素的集合。
Sorted sets: 类似Sets,但是每个字符串元素都关联到一个叫score浮动数值（floating number value）。里面的元素总是通过score进行着排序，所以不同的是，它是可以检索的一系列元素
Hashes: 由field和关联的value组成的map。field和value都是字符串的
